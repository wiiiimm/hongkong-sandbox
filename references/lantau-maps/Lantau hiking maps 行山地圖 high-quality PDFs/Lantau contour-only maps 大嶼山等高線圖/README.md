# Lantau contour-only map assets / 大嶼山等高線圖

Generated from the official Lands Department 5 m-grid Digital Terrain Model for Hong Kong. Outputs use 20 m contour intervals with heavier 100 m index contours. The PNG/PDF/SVG files are intentionally label-free for illustration workflows; a labelled reference PNG is included separately.

Files:
- `lantau-contour-only-20m-black-on-white.png/pdf/svg`
- `lantau-contour-only-20m-blueprint.png/pdf/svg`
- `lantau-contour-only-20m-labelled-reference.png`
- `contour-only-manifest.json`

Use case: contour-only base artwork for a side-view/island illustration.
