# Whole-Lantau Lantau Trail composite

OFCA does not publish a single whole-Lantau Lantau Trail PDF in its digital map index. This composite uses the Lands Department SS09 Lantau Island GeoPDF as the base and overlays official Lantau Trail section route geometries extracted from hiking.gov.hk.

Files:
- `whole-lantau-lantau-trail-composite.png`
- `whole-lantau-lantau-trail-composite.pdf`
- `lantau-trail-official-routes.geojson`

Use the original OFCA section PDFs for detailed route-by-route reference; this composite is an overview/planning map.
