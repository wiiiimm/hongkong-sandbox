# Lantau 3D Rendering

This folder contains a self-contained rotatable 3D terrain rendering of Lantau Island, Hong Kong.

It was generated by Codex from public DEM raster tiles. It is not a pre-existing uploaded 3D model.

## Contents

- `lantau-3d-viewer.html` - interactive browser viewer built with Three.js and OrbitControls.
- `lantau-terrain-mesh.json` - local mesh data loaded by the browser viewer.
- `lantau-terrain.obj` - importable OBJ terrain mesh for 3D tools.
- `lantau-3d-viewer-desktop.png` - verified desktop screenshot.
- `lantau-3d-viewer-mobile.png` - verified mobile screenshot.
- `generate_lantau_3d.py` - script that creates the mesh, OBJ, and viewer.
- `generate_lantau_skyline.py` - shared DEM/tile helpers and the approximate Lantau coastline mask.
- `verify_3d_viewer.cjs` - Playwright verification script.
- `vendor/` - local Three.js browser modules used by the viewer.
- `dem_tiles/` - cached public Terrarium DEM tiles used to generate the mesh.

## Data Source

The terrain source is the AWS Registry of Open Data Terrain Tiles dataset:

`https://registry.opendata.aws/terrain-tiles/`

The script uses the public S3 tile URL pattern:

```text
https://s3.amazonaws.com/elevation-tiles-prod/terrarium/{z}/{x}/{y}.png
```

For this model, tiles were cached at zoom `13` in `dem_tiles/`.

Terrarium PNG tiles encode elevation in RGB. The decode formula used by the generator is:

```text
(red * 256 + green + blue / 256) - 32768
```

## Generation Method

1. Load cached Terrarium DEM tiles around Lantau.
2. Decode each tile pixel into elevation metres.
3. Apply the approximate Lantau coastline polygon embedded in `generate_lantau_skyline.py`.
4. Sample the island into a `220 x 140` grid.
5. Convert longitude/latitude samples into local metre coordinates:
   - `x` = west-east distance
   - `z` = south-north distance
   - `y` = elevation
6. Apply `2.0x` vertical exaggeration for legibility in the viewer.
7. Export:
   - `lantau-terrain-mesh.json`
   - `lantau-terrain.obj`
   - `lantau-3d-viewer.html`

From the project root:

```sh
/Users/williamli/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 -m http.server 4173 --directory codex
```

Then open:

```text
http://127.0.0.1:4173/lantau-3d-rendering/lantau-3d-viewer.html
```

The viewer is self-contained relative to this folder. It loads:

- `./lantau-terrain-mesh.json`
- `./vendor/three.module.js`
- `./vendor/OrbitControls.js`

From the project root:


## Caveats

- This is based on public global DEM raster tiles, not official Hong Kong 5 m or 20 m DEM.
- The coastline mask is approximate and hand-coded.
- The mesh excludes buildings, vegetation, bridges, and man-made structures.
- The viewer uses vertical exaggeration for readability, so the visual height is not 1:1 with horizontal scale.
- This is suitable for terrain visualisation, art direction, and downstream illustration references, not survey-grade analysis.
