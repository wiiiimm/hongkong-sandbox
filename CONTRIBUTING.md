# Contributing to Hong Kong Sandbox

Thanks for wanting to improve it! 🇭🇰

## How to contribute

- The app is **static, no build step** — run it locally with `node tools/dev-server.mjs`
  (or any static server) and refresh. See the README.
- Keep changes focused; use semantic commit messages (`feat:`, `fix:`, `docs:`, …).
- Sign off your commits: `git commit -s` (Developer Certificate of Origin).
