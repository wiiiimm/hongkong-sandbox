# Cathay Pacific livery source artwork

These files are build inputs for `../trim_a350_glb.mjs` and
`../trim_777_glb.mjs`; they are not loaded by the viewer at runtime.

- `cathay-brushwing.svg` — complete Cathay brushwing mark, supplied by the
  project owner on 2026-07-12 as `0293.HK.svg`. The build uses it in white on
  the A350 tail and inward winglet faces, in white on the 777 tail, and in jade
  beside each cockpit.
- `cathay-wordmark.jpg` — Cathay Pacific lock-up supplied by the project owner
  on 2026-07-12 as `Cathay_Pacific.renditionimage.450.175.jpg`. The build crops
  out the brushwing and uses only the `CATHAY PACIFIC` lettering as a mask.

The project owner also supplied five Boeing 777 reference photographs on
2026-07-12, covering both sides, the tail and the cockpit brushwing. They were
used to check the 777 placement and proportions but are not redistributed here.

A350 placement and the pale-jade lower-fuselage treatment were checked against
these Cathay Pacific reference images (downloaded by the project owner on
2026-07-12):

- `A350-1000 new features EN-1.renditionimage.1700.1133.jpg` —
  <https://www.cathaypacific.com/content/dam/focal-point/cx/inspiration/missing-images/A350-1000%20new%20features%20EN-1.renditionimage.1700.1133.jpg>
- `Hero-CBC-1.renditionimage.1700.600.jpg` —
  <https://www.cathaypacific.com/content/dam/focal-point/cx/inspiration/2018/07/Hero-CBC-1.renditionimage.1700.600.jpg>
